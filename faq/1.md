*Otázky*
--------

1. Jaký programovací jazyk a v jaké verzi vaše aplikace využívá?
2. Jaký framework (pokud vůbec) a v jaké verzi je použit pro tvorbu webové aplikace?
3. Jakého správce balíčků používáte v rámci vaší aplikace?
4. Jaký typ databáze aplikace vužívá?
5. Jak vypadá connection string do databáze?

*Odpovědi*
----------

1. Programovací jazyk: Python 3 (kód je napsaný pro Python 3.x a spouští se přes `python app.py`).
2. Framework: Flask (mikro‑framework pro webové aplikace; použit přímo v `app.py`).
3. Správce balíčků: pip / virtuální prostředí (v repo není `requirements.txt`, doporučuji `pip` a virtuální prostředí vázat na `Flask`).
4. Databáze: SQLite (lokální souborová databáze, používá modul `sqlite3`).
5. Connection string / způsob připojení: aplikace používá `sqlite3.connect('ptaci.db')`. Pokud byste používali SQLAlchemy, odpovídající URI by byl `sqlite:///ptaci.db`.
