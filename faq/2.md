*Otázky*
--------

1. Napište vědecký název veškerého ptactva, které splňuje současně následující kritéria: vyskytuje se v Evropě, je téméř ohrožené a jsou hmyzožravci.
2. Napište SQL dotaz, kterým lze získat informaci z předchozí otázky. Budete muset najít způsob, jak takový dotaz získat. Do odpovědi nevkládejte dotaz, který obsahuje proměnné.
3. Jak se do HTML šablony dostanou data o ptactvu, která si uživatel nechal vypsat s použitím filtru? Do odpovědi vložte volání funkce, kde se tak děje.
4. Jakou JS knihovnu používáte pro zobrazování grafů? Odkud se stahuje?

*Odpovědi*
----------

1. Vědecké názvy druhů splňujících podmínky (Evropa, "Téměř ohrožený", hmyzožravec):
	- Coracias garrulus
	- Vanellus vanellus
	- Numenius arquata

2. SQL dotaz bez proměnných, který vrátí požadované vědecké názvy:
	SELECT vedecky_nazev FROM ptaci WHERE vyskyt_kontinent = 'Evropa' AND status_ohrozeni = 'Téměř ohrožený' AND typ_potravy = 'hmyzožravec';

3. Data filtrovaná uživatelem jsou předána do HTML šablony voláním (v `app.py`):
	return render_template('dashboard.html', ptaci=ptaci, filter_options=filter_options, filters=request.args, stats=stats, graf_rad_labels=graf_rad_labels, graf_rad_data=graf_rad_data, graf_potrava_labels=graf_potrava_labels, graf_potrava_data=graf_potrava_data, graf_migrace_labels=graf_migrace_labels, graf_migrace_data=graf_migrace_data, graf_kontinent_labels=graf_kontinent_labels, graf_kontinent_data=graf_kontinent_data)

4. JavaScript knihovna pro grafy: Chart.js (verze v šabloně `4.4.7`), načítá se z CDN jsDelivr:
	https://cdn.jsdelivr.net/npm/chart.js@4.4.7/dist/chart.umd.min.js
